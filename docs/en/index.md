# VocabQuiz Documentation

VocabQuiz is an Android vocabulary quiz app built with Kotlin and Jetpack Compose. It connects to Google Sheets for vocabulary data and uses Google Sign-In for authentication.

## Table of Contents
* TOC
{:toc}

## Getting started

- Follow the setup guide to sign in and connect your Sheets data.
- Use the quiz screen to practice, reveal answers, and navigate cards.
- Use Settings to choose the spreadsheet, sheet tab, UI language, and see supported language names.
